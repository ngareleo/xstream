# Client Conventions

- `useLazyLoadQuery` lives **only** in `src/pages/`. Components read data via fragments + `useFragment`.
- Fragment naming: `<ComponentName>_<propName>` (e.g. `VideoCard_video`). Operation names must start with the containing filename (relay-compiler enforces).
- Component definition style: `export const Name: FC<Props> = (…) => { … };` — always `FC`. Never function declarations.
- Styles: Griffel (`makeStyles`) only. Classes in `global.css` are limited to browser globals (resets, fonts, scrollbar, `[data-tip]`, `body.resizing`).
- Nova eventing: bubble events from children via `useNovaEventing().bubble()`; intercept in parents via `NovaEventingInterceptor`. One `NovaEventingProvider` at the app root, never more. Events are colocated: `ComponentName.events.ts`.
- **`*.tsx` files house only React.** Component files contain the React component definition, props/types, fragments, mutations, and queries — nothing else. Constants, formatters, type aliases, and helper functions move to a colocated `ComponentName.utils.ts` (the fifth slot alongside `.styles.ts`, `.strings.ts`, `.events.ts`, `.stories.tsx`). Exception: a single-use closure helper that exists only to satisfy a callback signature (e.g. a tiny `setTimeout` handler) may stay inline. Multi-use or non-trivial helpers must move out. A second React component may live in the same `.tsx` only when it is a private rendering helper that has no reason to exist outside that file; if it has its own fragment + mutation, lift it to a sibling `.tsx`.
- User-visible strings use `react-localization` (`new LocalizedStrings({ en: { … } })`) — no plain string exports in `*.strings.ts`.
- Every component has a `<ComponentName>.stories.tsx`. Relay fragment components use the `withRelay` decorator from `client/src/storybook/withRelay.tsx` (not raw `createMockEnvironment`). Story queries carry `@relay_test_operation`.
- **Icons:** Import only from `~/lib/icons`. That file is a thin wrapper over `@heroicons/react@1.0.6` (Outline). Do not hand-roll SVGs — instead add a Heroicons re-export with the same naming pattern. Design-system source of truth: [Figma Admin System UI Kit (Community)](https://www.figma.com/design/zO44HkPod5kUESUofiCn8o/Admin-System-UI-Kit--Community-) "Heroicons Outline" frame. Note: no TV icon in Outline v1 (`DesktopComputerIcon` used as substitute); no `arrows-pointing-in` icon (kept hand-rolled with a v2 path until upgrading).
